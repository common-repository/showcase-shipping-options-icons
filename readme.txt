The Showcase Shipping Options (icons) plugin enables you to easily display shipping method icons anywhere on your WordPress website via a shortcode.

== Description ==

The Showcase Shipping Options (icons) plugin is a simple, lightweight plugin designed to allow you to display visually appealing shipping method icons on your website. Whether you run an e-commerce platform, a service-providing website, or an informative blog, this plugin offers a streamlined way to show the shipping options you offer.

Features:
- Supports multiple shipping method icons such as UPS, DHL, FedEx, and many more.
- Simple settings page within the WordPress admin to choose which icons you want to display.
- Shortcode support for flexible placement on pages, posts, or widgets.
- Customizable icon sizes (in pixels) for seamless integration with your site design.

== Installation ==

1. Upload the shipping-icons-plugin folder to the /wp-content/plugins/ directory, or install the plugin directly via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Shipping Icons settings page to configure which icons are displayed.
4. Add the shortcode [ssopm_PM] to any page, post, or widget where you want the icons to appear.

== Frequently Asked Questions ==

= Can I adjust the size of the icons? =

Yes, you can adjust the size of the shipping method icons in pixels. The default is set to 65px.

= Can I adjust the spacing between the icons? =

Yes, you can adjust the spacing between the shipping method icons in pixels. The default is set to 5px.

= What's the shape of the icons? =

You can choose between rectangle and square icons.

= What shipping method icons are included? =

We currently have icons for the following: An Post, Asendia, Australia Post, Austrian Post, BG Post, Bpost, Brazil Correios, Bring, Canada Post, China Post, Colis Privé, Colissimo, Correos, Correos Express, Cyprus Post, Deutsche Post, DHL, DPD, DSV, ELTA, Emirates Post, Estafeta, FedEx, GLS, Hermes, Hellenic Post, Hong Kong Post, Hrvatska Pošta, India Post, Itella, Japan Post, Korea Post, La Poste, Lietuvos Paštas, Magyar Posta, Malaysia Post, Mondial Relay, Nacex, New Zealand Post, Österreichische Post, Polska Posta, Poste Italiane, Posten Norge, PostNL, PostNord, PostNord Sverige, Royal Mail, Russian Post, Seur, Singapore Post, Swiss Post, Thailand Post, TNT, UK Mail, UPS, USPS, Vietnam Post, Yamato Japan, and Yodel.

= Does this plugin work with every WordPress theme? =

The plugin is designed to work with most themes, but we cannot guarantee perfect compatibility due to the diversity of themes available.
